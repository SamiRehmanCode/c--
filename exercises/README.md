# Practice Exercises

This folder contains programming exercises to practice system programming concepts.

## Exercise Categories

### 1. Basic System Calls

- File operations (open, read, write, close)
- Process information (getpid, getppid, getuid)
- Directory operations (opendir, readdir, closedir)

### 2. Process Management

- Process creation with fork()
- Process execution with exec family
- Process synchronization with wait()
- Zombie and orphan processes

### 3. Threading

- Thread creation and termination
- Thread synchronization (mutexes, condition variables)
- Producer-consumer problems
- Reader-writer problems

### 4. Inter-Process Communication

- Pipes (anonymous and named)
- Shared memory
- Message queues
- Semaphores

### 5. Signal Handling

- Signal registration
- Signal handling functions
- Signal masking and blocking

### 6. Network Programming

- Socket creation and management
- TCP/UDP clients and servers
- Concurrent server design

## Exercise Templates

Each exercise should include:

1. Problem description
2. Sample input/output
3. Skeleton code to start with
4. Test cases
5. Solution (in separate folder)

## How to Use

1. Read the problem description carefully
2. Implement your solution
3. Test with provided test cases
4. Compare with reference solution
5. Optimize and improve your code
